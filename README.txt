欢迎使用 “物理电学模拟器”！

=====================

使用说明

=====================

0. 概述

本程序能够解决高中物理的电学实验，能够轻松处理串联、并联等简单电路，并能分析惠斯通电桥等复杂电路问题。

本程序使用pygame进行用户界面、电路图的渲染与事件处理。

本程序的电路分析算法基于深度优先搜索生成路径，并综合运用基尔霍夫电压/电流定律，生成以电流为未知数、关于电动势与电压降的多元一次方程组，并利用numpy的最小二乘估计算出各个元件的电流值，反馈到各个元件，并利用部分电路的欧姆定律得出单个元件的电压降，实现了复杂电路问题的仿真模拟。


1. 界面

本程序的主界面包括左侧的【元件面板】、中间的【工作区】、右侧的【属性面板】三部分。

【元件面板】：左侧的窄条形区域，上有各个元件的按钮。

【工作区】：中间区域，显示元件电路图的主要区域。

【属性面板】：显示当前选中的元件属性。


2. 开始使用

在左侧的【元件面板】任意按钮上按下鼠标并拖拽即可拖出元件，移至中间工作区后松开鼠标即可放置元件。

点击并拖拽【工作区】内的任意元件均可改变其位置，将其拖至【元件面板】则将删除这一元件。

将鼠标悬停在【工作区】内的任意元件左右两端，将显示红色端点，此时按下鼠标并拖拽将引出导线，将导线拉动至另一元件的端点松手即可连线。

在您添加/修改元件或连线时，电路会自动进行分析。


左下角的红色按钮可以清空工作区内的所有元件。

3. 属性修改

单击【工作区】内的任意元件，其将被选中，并在右侧属性栏显示各项属性的名称及数值（例如：电压、电流等）。

部分属性为元件的常量，不可修改；可修改的属性将显示“+”“-”按钮。

4. 元件特殊属性

【开关】：双击可以改变开关的开闭状态。

【滑动变阻器】：在滑动变阻器的小箭头处拖拽可改变滑动电阻器的有效部分。

【电阻箱】：在属性面板中可以显示并调整电阻箱的阻值。由于电阻箱的阻值有较大的调节分度，建议在您欲使用较大电阻时使用电阻箱来替代。
